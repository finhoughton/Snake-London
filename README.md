# Snake: London

A jet lag game played on the TFL network. Teams physically travel the tube, claiming stations by completing challenges at interchanges. Each team controls a snake that grows through the network — if your snake's vulnerable **neck** crosses another team's claimed territory, you crash and lose.

see full rules: https://docs.google.com/document/d/1heYIjxtCtdxrPid3nXtNOxfgfdhEb5264-mMFSi3dh4/edit?tab=t.0#heading=h.y9ay5dm7gnlq

Currently, this repo is the game engine and map renderer.

## Game concepts

| Term            | Meaning                                                         |
| --------------- | --------------------------------------------------------------- |
| **Interchange** | A station on the game map (some combine multiple real stations) |
| **Origin**      | A team's fixed starting interchange                             |
| **Anchor**      | The last interchange where a challenge was completed            |
| **Front**       | The interchange where a challenge is currently being attempted  |
| **Body**        | Permanently claimed path from Origin → Anchor                   |
| **Neck**        | Vulnerable path from Anchor → Front (active during a challenge) |
| **Crash**       | Occurs when the neck enters another team's claimed territory    |

## Project structure

```
map/
  connections.json   network topology — lines, stations, adjacency
  geometry.json      station centre coordinates in SVG space (generated by build_geometry.py)
  snake map.svg      hand-designed map

map.py               graph model of the tube network
game.py              game state machine — challenge flow, crash detection, win conditions
render.py            SVG/PNG renderer — overlays team colours onto the base map
build_geometry.py    extracts station positions from the SVG into geometry.json
challenges.py        challenge pool and difficulty scaling
example.py           example game script
tests/               pytest suite
```

## Usage

```python
from game import new_game
from render import render_map, svg_to_png

game = new_game(start_positions={
    "Alpha": "Baker Street",
    "Beta":  "Stratford",
})

game.initial_request_challenge("Alpha")
game.complete_challenge("Alpha", "Jubilee")

game.request_challenge("Alpha", "Bond Street")
# ... etc

render_map(game, "current_map.svg")
svg_to_png("current_map.svg", "current_map.png")
```

See `example.py` for a full three-team game.

## Requirements

Python ≥ 3.11, `resvg-py`. Tests: `pytest`.
